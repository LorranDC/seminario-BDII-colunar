# seminario-BDII-colunar
Testes práticos para descrever o comportamento do Banco de Dados orientado à linhas versus o comportamento do Banco de Dados Colunar

PRIMEIRO EXPERIMENTO
  Simulação visual com HTML/CSS/JS/Python com animações para exemplificar a movimentação padrão de um SGBD em relação a um Banco de Dados Linear e a um Banco de Dados Colunar, com algumas operações CRUD exemplificadas

SEGUNDO EXPERIMENTO
  Código em Python a fim de realizar a comparação de operações executadas em SQLite(Linear) e DuckDB Parquet(Colunar) em cima de um volume massivo de registros a fim de comparar em qual situação cada tipo de Banco de Dados se sobressai
